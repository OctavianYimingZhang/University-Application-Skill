# Submission Readiness

Use this focused Skill for final pre-submission checks.

## Workflow

1. Read [`../../references/submission.md`](../../references/submission.md), [`../../references/quality-checks.md`](../../references/quality-checks.md), and [`../../references/evidence-contract.md`](../../references/evidence-contract.md).
2. Confirm programme, intake, citizenship, destination, funding basis, and source policy when those fields affect submission or visa readiness.
3. Check programme selection, documents, language tests, references, fees, scholarships, deadlines, and visa-sensitive items.
4. Separate blockers from optional improvements.
5. Mark each item with a normalized evidence record or a gap. Only records that pass the evidence invariant may satisfy a readiness item.
6. Return a final action list without implying submission has occurred.

## Output

Produce a concise readiness summary, blockers, date-sensitive tasks, and source log.

Default to English. Use another output language only when the user explicitly requests it.
