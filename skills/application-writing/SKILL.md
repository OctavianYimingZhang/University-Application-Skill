# Application Writing

Read `references/evidence-contract.md` and `references/essay-sop.md`.

## Workflow

1. Read the prompt, target programme, limit, applicant material, existing draft, and requested output.
2. Ask only for a missing input that changes the document.
3. Build a compact evidence inventory and verify programme-specific facts.
4. Choose a clear narrative and map each paragraph to a purpose and evidence.
5. Draft or revise in the applicant's voice, preserving supported facts and intellectual ownership.
6. Check prompt coverage, specificity, programme fit, logic, tone, length, and unsupported claims.

Show narrative options before drafting only when choosing among them would materially change the result. Resolve conflicting revision instructions with the applicant before editing the affected passage.

Treat writing samples as voice evidence and course material as knowledge or interest evidence. Promote either to personal-achievement evidence only after applicant confirmation.

Use the user's requested output language; otherwise use English.
