# Quality Checks

Block final recommendations when:

- a required source URL is missing
- a deadline is stale or undated
- language requirements conflict across official pages
- fee or scholarship figures lack a source date
- visa guidance is requested but citizenship or destination is unknown
- eligibility depends on grades, prerequisites, portfolio, work experience, or professional accreditation that has not been verified
- an applicant comparison relies on evidence that does not pass [`evidence-contract.md`](evidence-contract.md)
- an official fact relies only on a local document or user confirmation
- a materials or submission item relies only on user confirmation
- an admissions draft has a pending, missing, conflicted, or unlocated revision-ledger item
- a source is available but the extracted fact is unverified, conflicted, partial, stale, or for another application cycle
- visa readiness lacks current official government or immigration-authority provenance
